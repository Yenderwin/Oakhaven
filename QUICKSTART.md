# Oakhaven - Guía Rápida de Inicio

## ¿Qué es Oakhaven?

Oakhaven es un juego de misterio interactivo de 150 páginas donde descubrirás un diario antiguo con secretos ocultos, códigos para descodificar y revelaciones que cuestionarán tu realidad.

## Instalación Rápida

### Opción 1: Aplicación Electron (Recomendado)

```bash
# 1. Clonar repositorio
git clone https://github.com/Yenderwin/Oakhaven.git
cd Oakhaven

# 2. Instalar dependencias
npm install

# 3. Ejecutar aplicación
npm start
```

### Opción 2: Navegador Web

```bash
# 1. Instalar dependencias
npm install

# 2. Construir archivos
npm run build

# 3. Abrir index.html en tu navegador
open index.html
```

## Cómo Jugar

### Fase 1: Descubrimiento (Páginas 1-50)
- Lee cuidadosamente cada página
- Observa si hay elementos destacados
- Busca códigos ocultos en los márgenes

### Fase 2: Investigación (Páginas 51-100)
- La trama se vuelve más intensa
- Presta atención a detalles repetidos
- Las pistas se conectan de formas inesperadas

### Fase 3: Revelación (Páginas 101-150)
- Todo cambia
- Usa los botones especiales para revelar secretos
- ¡La verdad te espera!

## Elementos Especiales

### 👻 Tinta Invisible
Algunas páginas contienen texto oculto que solo aparece con luz UV.
- **Cómo revelar**: Presiona `R` o haz clic en "👻 Revelar Secreto"
- **Ubicaciones**: Páginas 23, 45, 75, 120, 150
- **Efecto**: Destello amarillo cuando se revela

### 🔐 Códigos Cifrados
Existen códigos que debes descodificar.
- **Tipos**: César, Numérico, Símbolos
- **Cómo descodificar**: Presiona `D` o haz clic en "🔐 Descodificar"
- **Pista**: Lee con cuidado las instrucciones en la página

### 📝 Páginas Rayadas
Representan intentos de borrar información.
- **Significado**: Alguien intentó ocultar algo importante
- **Acción**: Lee entre las líneas

## Guardado de Progreso

Tu progreso se guarda automáticamente cada 5 páginas.

Para guardar manualmente:
1. Presiona `ESC` para pausar
2. Haz clic en "Guardar Progreso"

## Consejos para Jugar

1. **Lee lentamente** - Cada palabra puede ser importante
2. **Toma notas** - Anota números, patrones, conexiones
3. **Observa los detalles** - Los márgenes, los espacios en blanco
4. **No te apresures** - Este juego está diseñado para ser inquietante
5. **Siente la atmósfera** - La música y los gráficos tienen un propósito

## Preguntas Frecuentes

**P: ¿Cuánto tiempo toma completar el juego?**
R: Aproximadamente 2-3 horas si lees atentamente.

**P: ¿Hay múltiples finales?**
R: El final es el mismo, pero tu interpretación puede variar.

**P: ¿Qué pasa si pierdo mis datos?**
R: Puedes recuperar tu última guardada automática.

**P: ¿Es realmente terrorífico?**
R: Más psicológico que gore. Es inquietante y desconcertante.

**P: ¿Necesito jugar en la oscuridad?**
R: No es necesario, pero se recomienda para la atmósfera completa.
